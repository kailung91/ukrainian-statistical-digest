# Ukrainian Statistical Digest

Інтерактивний SPA-застосунок для візуалізації статистичних даних Держстату України. Проєкт генерує хороплет-карти та друковані звіти (формат А4) на основі PDF та CSV файлів.

## Архітектура та стек
* Фреймворк: React 18, Vite 7.
* Обробка PDF: Generative AI API (Gemini як основний, Claude як резервний).
* Обробка CSV: Локальний клієнтський парсинг (PapaParse) з нормалізацією кодів КАТОТТГ.
* Візуалізація: Кастомні SVG-полігони (25 канонічних регіонів, кастомний color ramp).
* Архітектура UI: Компоненти без стану (dumb components), бізнес-логіка та обчислення похідних даних зосереджені в `App.jsx`.

## Локальний запуск
1. Виконати `npm install`.
2. Створити файл `.env.local` та додати ключі:
   VITE_GEMINI_API_KEY=твій_ключ
   VITE_CLAUDE_API_KEY=твій_ключ
3. Запустити сервер розробки: `npm run dev`.

## Особливості розробки
* Заборонено використовувати зовнішні бібліотеки карт (Leaflet/Mapbox) для забезпечення портабельності та якісного друку.
* Усі стилі прописуються inline, без Tailwind чи CSS-модулів.
* Робота з AI-провайдерами винесена в ізольовані сервіси (`src/services/providers/`).
